
# 我的电视

电视直播软件，安装即可使用

## 使用

1. 下载
    * 下载本项目中的 `my-tv-1.apk` 文件

2. 安裝
    * U盘安装
    * 小米电视可以使用小米电视助手进行安装
    * 如电视可以启用ADB，也可以通过ADB进行安装
       ```shell
       adb install my-tv.apk
